# de-biasing group project IB 2021

## dependencies

Python modules:

- gensim 4.0.b0
- nltk
- adjustText
- seaborn
- sklearn
- numpy
- matplotlib
- pandas

You also need to download [json-simple.jar](https://www.javatpoint.com/jsonpages/json-simple-1.1.1.jar), rename it to `json-simple.jar`, and place it in the `app/api/` directory.
